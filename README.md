# Task-4-Sentiment Analysis using Machine Learning
Sentiment Analysis using Machine Learning and NLP to classify Google Play Store user reviews into Positive, Negative, and Neutral sentiments using Python and Scikit-learn.
## 📌 Project Overview

This project focuses on **Sentiment Analysis** using **Natural Language Processing (NLP)** and **Machine Learning**. The objective is to classify Google Play Store user reviews into **Positive**, **Negative**, and **Neutral** sentiments based on review text.

The project includes text preprocessing, TF-IDF vectorization, model training using the Multinomial Naive Bayes algorithm, model evaluation, and custom sentiment prediction.

---

## 📂 Dataset

**Dataset:** Google Play Store User Reviews

The dataset contains:
- App Name
- User Reviews
- Sentiment Labels
- Sentiment Polarity
- Sentiment Subjectivity

## 📊 Project Workflow

### 1. Data Loading
- Loaded the Google Play Store User Reviews dataset.

### 2. Data Cleaning
- Checked missing values.
- Removed null records.
- Selected required columns for analysis.

### 3. Exploratory Data Analysis
- Analyzed sentiment distribution.
- Visualized review categories.

### 4. Text Preprocessing
- Encoded sentiment labels.
- Converted review text into numerical features using TF-IDF Vectorization.

### 5. Model Building
- Split the dataset into training and testing sets.
- Trained a Multinomial Naive Bayes classifier.

### 6. Model Evaluation
- Evaluated model performance using Accuracy Score.
- Generated Confusion Matrix.
- Generated Classification Report.

### 7. Custom Prediction
- Predicted sentiment for user-entered review text.

---

## 📈 Results

- Model Accuracy: **68.96%**
- Successfully classified user reviews into Positive, Negative, and Neutral sentiments.
